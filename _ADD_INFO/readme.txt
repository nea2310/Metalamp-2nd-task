
 Описание файловой структуры проекта: 
 
 1) _ADD_INFO  == справка, заметки
 
 2) dist  == собранная статическая вёрстка, готовое приложение
 
 3) src  == исходный код проекта
 
 4) assets  == шрифты, изображения, дополнительные файлы
 
 5) components  == отдельные элементы страницы, части, куски, модули. 
 
 6) pages  == страницы сайта
 
 7) plugins  == сторонние плагины, слайдеры, календари, Джеквери и тп. 
 
 8) devDependencies - модули которые используються для сборки.
    dependencies - модули которые используються в коде.
 
 
-------------------------------------------------------------------------

 1) файлы стилей для каждого компонента и страницы свои, в этой же папке будет лежать файл стилей для медиаЗапросов с именем  media_name.scss
 
 2) все стили, плагины и тп, импортируються в файле style.scss
 
 3) все импорты JS файлов в одном файле javascript.js который будет лежать в папке ассетс , js .. а он в свою очеред будет импортирован в index.js
